Basic Functions
Prevents invalid input (decimals only, button does nothing when no input is entered).
Warns when bad values are entered (over 13oz, over 3.5 for letters).

Extra Credit
Tabs ("Calculate" tab and "Info" tab).
Input remains intact when rotating the device and switching tabs.
Both images and text inside info area.
Scrollable info + Toast notification reminding the user to scroll to see all info.
